# Beauty Memory Game - Palcare Laboratories

This is the complete source code for the Beauty Memory Game.

## Requirements
- **Node.js**: You must have Node.js installed on the computer. Download it from [nodejs.org](https://nodejs.org/).

## How to Install & Run

1.  **Install Dependencies** (First time only):
    - Double-click `install_dependencies.bat`.
    - Wait for the window to close.

2.  **Start the Game**:
    - Double-click `start_game.bat`.
    - The game will automatically open in your default web browser at `http://localhost:3000`.

## Player Data
- Player information (Name, Email, Phone) is saved to `players-data.xlsx` in this folder.
- **Important**: Ensure `players-data.xlsx` is CLOSED before starting the game, or data cannot be saved.

## Troubleshooting
- If the game doesn't start, ensure Node.js is installed.
- Check if port 3000 is already in use.
- If data is not saving, check if the Excel file is open.
